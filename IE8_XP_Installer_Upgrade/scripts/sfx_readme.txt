7zSD.sfx is 7-Zip SFX (self-extracting) module for installers.
The version included here is 16.04

The module is part of LZMA SDK <http://www.7-zip.org/a/lzma1604.7z>
LZMA SDK is written and placed in the public domain by Igor Pavlov.

7zSD_upxed.sfx is the UPX compressed version of the aforementioned
module. UPX <https://upx.github.io/>
The command is invoked like this:
upx --ultra-brute -o7zSD_upxed.sfx 7zSD.sfx
